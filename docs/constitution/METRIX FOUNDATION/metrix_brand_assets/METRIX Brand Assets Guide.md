# METRIX Brand Assets

Version: 1.0  
Status: Working master asset pack

## Primary assets

- `metrix-wordmark.svg` — primary wordmark using `currentColor`
- `metrix-wordmark-white.svg` — white wordmark
- `metrix-wordmark-black.svg` — black wordmark
- `metrix-icon.svg` — app icon / favicon master
- `favicon.ico` — browser favicon
- `metrix-icon-192.png` — PWA icon
- `metrix-icon-512.png` — app icon

## Usage rules

1. Use the SVG wordmark wherever possible.
2. Do not stretch, skew, recolor, outline, or add shadows outside the approved asset.
3. Minimum digital wordmark height: 16px.
4. Clear space around the wordmark should be at least the height of the letter `M`.
5. Prefer high-contrast usage: white on dark, black on light.
6. For the METRIX chat page header, use `metrix-wordmark.svg` and control color with CSS `currentColor`.
